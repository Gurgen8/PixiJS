# Bomberman

## Описание
Классическая аркадная игра «Bomberman», где игрок должен перемещаться по сетке, устанавливать бомбы для уничтожения препятствий и врагов, а также собирать улучшения.

## Архитектура
Структура директории `src/games/bomberman/`:

- **GameManager.ts**: Центральный контроллер игровой логики. Отслеживает очки (score), жизни игрока (lives) и параметры бомб (capacity, range).
- **scenes/**: 
  - `BombermanGameScene`: Основной цикл игры, управление игроком, сеткой и бомбами.
- **entities/**: Игровые сущности.
  - `Player`: Игрок, логика перемещения, получение урона.
  - `Enemy`: ИИ врагов (например, базовый патруль или преследование).
  - `Explosion` / `Bomb` / `GridSystem`: Логика бомб, взрывов и привязка к клеточной системе (Grid).
- **systems/**:
  - `BombManager`: Отслеживает установленные бомбы и управляет их детонацией.
- **ui/**: 
  - `HUD`: Отображает текущие характеристики игрока (жизни, очки, уровень и параметры бомб).
