# Fruit Collector

## Описание
Аркадная игра на реакцию, в которой игрок управляет корзиной, ловит падающие фрукты и старается избегать бомб. Игра разделена на волны с постепенно возрастающей интенсивностью и сложностью.

## Архитектура
Структура директории `src/games/fruitcollector/`:

- **GameManager.ts**: Управляет счетом, жизнями и текущей волной. Реагирует на события (пропущенные фрукты, пойманные бомбы).
- **scenes/**: 
  - `FruitCollectorGameScene`: Главная игровая сцена, объединяющая все системы.
- **entities/**: Игровые сущности.
  - `Basket`: Корзина, управляется игроком (влево/вправо).
  - Падающие объекты (Фрукты, Бомбы).
- **systems/**:
  - `ItemSpawner`: Система генерации. Отвечает за спавн объектов с неба, увеличивает интенсивность спавна с каждой новой волной.
- **ui/**: 
  - `HUD`: Отображает текущий счет, оставшиеся жизни и номер волны.
